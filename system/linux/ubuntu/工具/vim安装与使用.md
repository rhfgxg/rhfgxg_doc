## 安装 vim
使用下面的命令
```bash
sudo apt install vim
```

## vim 的基本使用
vim 有三个模式 普通模式、插入模式（编辑模式）, 命令模式
下面用一个例子演示

### 创建或打开文件
打开一个现有文件或创建一个新文件，进入普通模式
```bash
vim file_name    # 当文件不存在时，会自动创建新文件（左下角有 new file 提示）
```

### 普通模式
普通模式下，左下角显示
```bash
name.txt (New File)   # 提示为新文件
```
普通模式下的指令：
| 快捷键 | 作用 |
|------|------|
| G | G |
| n G | 将光标移至第n行行首 |
| n + | 将光标下移n行 |
| n - | 将光标上移n行 |
| n $ | 光标下移至第n行行尾 |
| 0(数字零) | 将光标移到所在行的行首 |
| $ | 将光标移到所在行的行尾 |
| ^ | 将光标移动所在行的第一个字符（非空字符） |
| h,j,k,l | 分别用于光标左移，下移，上移，右移一个字符 |
| H | 将光标移至当前屏幕首行的行首 |
| M | 将光标移至屏幕显示中间行的行首 |
| L | 将光标移至当前屏幕最低行的行首 |
| x | 删除光标处的字符 |
| dw | 删除一个单词 |
| dd | 删除光标所在行 |
| yy | 复制光标所在行 |
| p | 粘贴 |
| /str | 向右进行搜索 |
| ?str | 向左进行搜索 |
| n | 向相同的方向查找 （执行 /str 后向前或向后） |
|  | 向相反的方向查找 |
| :q |  退出 vim， 在对文件作出修改后不能直接使用 q 退出 |
| :q! | 强制退出 vim，不保存修改 |
| :w |  保存文件 |
| :Wq | 保存并退出 |
| :wq! | 强制保存并退出 |

### 进入插入模式
在普通模式下，按a, A, i, I, o, O 之一进入插入模式
| 快捷键 | 作用 |
|------|------|
| i | 光标不动, 输入文本的时候,在光标 前面 插入文本内容 |
| a | 光标向后移动一位, 输入文本的时候,在原光标 后面 插入文本内容 |
| I | 光标会移动该行的起始位置, 插入文本的时候,在行首位置开始插入内容 |
| A | 光标会移动该行的末尾位置, 插入文本的时候,在行末尾置开始插入内容 |
| o | 在此行的下一行 插入空行，光标移动到下一行 |
| O | 在此行的上一行 插入空行，光标移动到上一行 |

### 插入模式
插入模式下，右下角显示
```bash
INSERT
```
使用 ESC 退出 插入模式, 进入普通模式

### 保存修改的临时文件
在编辑的同时，vim 会在相同文件夹下 同时生成一个 .file_name.swp 的隐藏文件，用于异常退出时保存修改
在正常退出后，会将 .swp 文件删除
异常退出时，会保留 .swp 文件 下次打开次文件，会自动打开 .swp 文件，以恢复上次的修改
