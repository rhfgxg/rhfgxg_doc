## 安装及配置花生壳
原文链接：(花生壳5.0 for Linux使用教程-花生壳官网教程)[https://service.oray.com/question/11630.html]
(花生壳官方下载链接)[https://link.zhihu.com/?target=https%3A//hsk.oray.com/download]
选择 linux 复制命令，在终端右键粘贴
<img width="639" height="446" alt="image" src="https://github.com/user-attachments/assets/c479ba21-2ea6-491f-a693-110c0a3bf7a0" />

```bash
# 确保在用户目录下 下载花生壳压缩包
sudo wget "https://down.oray.com/hsk/linux/phddns_5.2.0_amd64.deb" -O phddns_5.2.0_amd64.deb

# 解压并安装 花生壳
sudo dpkg -i phddns-5.0.0-amd64.deb

# 查看花生壳支持的操作
sudo phddns
# start（启动）| status（状态）| stop（停止）| restart（重启）| reset（重置）| enable（设置开机自启动）| disable（关闭开机自启动）| version（查看版本）

# 使用花生壳的指令
sudo phddns 对应指令

# 这里我们只需要打开开机自启动
sudo phddns enable
```

## 登陆并激活内网穿透
浏览器访问http://b.oray.com，输入花生壳Linux 5.0在安装时产生SN码与默认登录密码admin登录
<img width="750" height="393" alt="image" src="https://github.com/user-attachments/assets/76a006a2-36e9-4151-8a0b-c5c4e89c7db8" />
首次登录，需先激活。提供扫码激活和密码激活两种方式，可灵活选择。
①扫码激活：需使用花生壳管理APP或微信进行扫码激活；
②密码激活：输入已注册的贝锐帐号密码激活。
激活成功后，进入花生壳管理平台，点击 增加映射
<img width="750" height="416" alt="image" src="https://github.com/user-attachments/assets/4f0bc3e1-ae0c-485b-9a69-35a7b5a37645" />
内网ip：在自己的终端使用
```bash
# 查看自己的内网 ip
sudo ifconfig 
# 返回内容中的 inet 192.168.43.**
# 192.168 开头的地址 
```
<img width="750" height="453" alt="image" src="https://github.com/user-attachments/assets/225e323c-9346-496f-b91e-f2ebb13cb0ac" />
1. 应用名称: 为此WEB应用命名
2. 应用图标: 可以给对应的服务选择一个图标，方便区分
3. 映射类型: 选择HTTPS
4. 外网域名: 选择在花生壳购买的域名或者刚刚注册账号赠送的域名
5. 外网端口: 443（默认）
6. 内网主机: 服务器内网IP地址
7. 内网端口: 所映射服务的内网端口：80 端口
8. 带宽 付费提升: 默认根据帐号等级分配相应的带宽，也可根据实际需求额外付费购买
9. 夜间带宽 付费提升: 18:00-次日8:00带宽速度提升100%，速度不低于5Mbps；带宽基数越大，夜间越快
